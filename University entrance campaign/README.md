У межах цього проєкту було реалізовано повний цикл побудови моделі машинного навчання для прогнозування зарахування абітурієнтів на основі їхніх результатів та наявності пільг. Було згенеровано штучний набір даних, який імітує реальний конкурсний відбір, виконано попередню обробку даних, масштабування ознак та поділ на навчальну і тестову вибірки.
Для розв’язання задачі було досліджено декілька архітектур нейронних мереж різної складності. Результати експериментів показали, що зі збільшенням кількості нейронів та шарів точність моделі зростає. Найкраща модель досягла accuracy близько 98%, що свідчить про здатність нейромережі ефективно відтворювати правила прийняття рішень, закладені у вихідних даних.
Отримана модель була використана для прогнозування результатів для всіх абітурієнтів та формування фінального списку зарахованих із урахуванням рейтингу та квот для пільгових категорій. Результати автоматично збережено у файл формату Excel.	
Загалом проєкт демонструє практичне застосування нейронних мереж для задач класифікації, а також показує, як машинне навчання може бути використане для підтримки прийняття рішень у сфері освіти. Реалізація може слугувати навчальним прикладом або основою для подальшого розширення, зокрема шляхом використання реальних даних, додавання нових факторів та ускладнення моделі.
